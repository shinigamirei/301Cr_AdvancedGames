Instructions for running/ playing game

upon running the exe in TicTacToe/Debug 
press 1 to launch the client or 2 to launch the server

the server will display the ip adress and take no other user input

copy the adress from the server and replace the one tin tictactoe.cpp with it in order to connect

you can select a move with the left mouse button, although there is no feedback

start a chat entry with t, submit with enter
the chat entry will apear on the server console
